Hi, This Set Permissions for Files and Directories

ubuntu@ip-172-31-94-148:/$ cd /var/www/html

  • To open the /var/www/html first, because we need to upload our local file to the /var/www/html first.

    • ubuntu@ip-172-31-94-148:/var/www/html$ ls -l

  • We also can use this codes to check what we have for now.

    • ubuntu@ip-172-31-94-148:/var/www/html$ sudo chown -R $root:$root /var/www/html

    ubuntu@ip-172-31-94-148:/var/www/html$ sudo chown -R $ubuntu:$ubuntu /var/www/html

  • Change owner of /var/www/html directory to apache user, here is uubuntu.

    • $ sudo chmod -R 777 /var/www/html

  • This permission can edit to transfer files to /var/www/html from our local files.
